# Формирование таблицы подтверждения рейсов

Программа сравнивает признаки выполнения рейса по данным оператора (столбец L)
и перевозчика (столбец AD), после чего создает Excel-файл с тремя листами:

- `На подтверждение` — оператор = 0, перевозчик = 1;
- `Сводка` — итоговые счетчики и сумма транспортной работы;
- `Ошибки проверки` — отсутствующие и недопустимые значения.

Дата берется из столбца T (`Транспортные сутки`), а если он пуст — из столбца C
(`Дата`). Остальные поля читаются по буквам B, E, L, Z, AC, AD и AF.

## Веб-приложение

1. Один раз запустите `install_dependencies.bat`.
2. Запустите `run_web.bat`.
3. В браузере выберите исходную таблицу и файл-шаблон.
4. Нажмите `Сформировать отчет`, затем скачайте готовый Excel.

Приложение доступно только локально по адресу `http://127.0.0.1:5000`.
Загруженные файлы не отправляются в интернет.

## Настольное приложение

1. Запустите `install_dependencies.bat`.
2. Запустите `run.bat`.
3. Выберите исходную таблицу, пример итоговой таблицы и папку сохранения.
4. Нажмите `Сформировать отчет`.

Результат сохраняется под именем
`Итоговая_таблица_подтверждение_рейсов.xlsx`.

## Запуск из командной строки

```text
py app.py --cli "путь\к\исходнику.xlsx" "путь\к\шаблону.csv" "папка результата"
```

Поддерживаемые входные форматы: `.xlsx` и `.csv`. CSV может быть в UTF-8 или
Windows-1251, разделитель определяется автоматически.
